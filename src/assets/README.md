## Assets

Usa esta carpeta para mantener tus recursos visuales separados del codigo:

- `placeholders/`: imagenes temporales para cards y secciones
- `icons/`: iconos o logotipos propios
- `images/`: fotos, mockups o capturas reales de proyectos

Reemplaza el archivo `placeholders/project-placeholder.svg` cuando tengas las imagenes reales de tu portafolio.
